# 终端配置代理

以 zsh 为例。

~vim ~/.zshrc

按下i，进入编辑模式，并添加如下配置：

# proxy list# 注意这里的端口号，要写成你自己的alias proxy='export all_proxy=socks5://127.0.0.1:7891'alias unproxy='unset all_proxy'

按下 :wq保存后退出。 输入下面的命令进行更新：

source ~/.zshrc

然后使用下面的命令开启代理

proxy

关闭代理的时候使用下面的命令

unproxy
